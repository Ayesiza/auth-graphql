# A login form  created with grapgql Mongodb Nodejs and React
# Clone
`git clone https://github.com/Ayesiza/auth-graphql.git`
# Installation
` nmp install`
# Run the Server
` npm run serve`
# Run Client
` npm start`
** Both client and server must be running to fetch data from sever side
